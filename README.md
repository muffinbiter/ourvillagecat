# Our Neighborhood Cat
A cozy pixel-art narrative village game

---

## Overview

**Our Neighborhood Cat** is a narrative exploration game where players control a wandering cat exploring a small village and interacting with its residents.

Each day the player spends limited action points visiting locations, talking with villagers, and making choices that influence the atmosphere of the town.

---

## Core Gameplay Loop

Explore village → Visit buildings →  
Interact with residents → Choose actions →  
Spend AP → Advance to the next day.

---

## Key Features

**Pixel Art World**  
A cozy village rendered in pixel graphics.

**Village Exploration**  
Seven buildings including café, bakery, flower shop, and blacksmith.

**NPC Interaction**  
Residents have distinct personalities and respond differently to player actions.

**AP System**  
Players manage daily action points to plan interactions.

**Village Mood System**  
Positive actions increase the village atmosphere.

**Narrative Ending**  
Player choices gradually warm the town and lead to different endings.

---

## Tech Stack

HTML5 / CSS / Vanilla JavaScript  
Single-file architecture with embedded assets  
Pixel-font rendering using neodgm

---

## Project Background

The project explores how **simple narrative interaction systems can create emotional engagement in small-scale simulation worlds**.
